Python正则表达式——re模块部分解析

发表于 2017-05-24 | 分类于 Python

  正则表达式，又称规则表达式。（英语：Regular Expression，在代码中常简写为regex、regexp或RE），计算机科学的一个概念。它是对字符串操作的一种逻辑公式，就是用事先定义好的一些特定字符、及这些特定字符的组合，组成一个“规则字符串”，这个“规则字符串”用来表达对字符串的一种过滤逻辑。因此它通常被用来检索、替换那些符合某个模式(规则)的文本。
  Python的正则表达式处理由re模块提供，本篇将讲解re模块中经常被用到的几个函数的基本用法。

  以下操作均在Python 3 GUI 中进行。
